# RADAR Training Platform — CLAUDE.md

## What This Is
A performance-based online course platform for real estate agents. Think Kajabi but purpose-built for RADAR (a seller identification system). Agents log in, complete lessons in order, and graduate when all lessons are done.

## Tech Stack
- **Next.js 14** (App Router, TypeScript)
- **Supabase** (auth + PostgreSQL for progress tracking)
- **Tailwind CSS** with custom brand tokens
- **No third-party UI library** — all components are custom

## Setup
```bash
cd platform
npm install
cp .env.local.example .env.local
# Fill in NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
# Run supabase/schema.sql in your Supabase SQL editor
npm run dev
```

## Key Architecture Decisions
- **Course content is static data** (TypeScript files in `/data/courses/`). No CMS needed.
- **Only progress is stored in Supabase** — what's completed, scores, graduation timestamp.
- **Lessons unlock sequentially** — each lesson requires the previous one to be complete.
- **Lesson types**: video, flashcard, reflection, quiz, roleplay (scored, must pass ≥ 8/10)

## Adding a New Course
1. Create `src/data/courses/ppc-plus.ts` following the same structure as `radar.ts`
2. Import and add it to `allCourses` in `src/data/courses/index.ts`
3. It will automatically appear on the dashboard

## Adding Images for Lessons
Images live in `platform/public/images/`. Reference them as `/images/filename.png` in the `image` field on any `Lesson` or `Course` object in the data file. Images are used:
- **Course card** (`coverImage` on Course) → thumbnail on dashboard + hero banner on course overview
- **Lesson** (`image` on Lesson) → thumbnail in lesson list, backdrop in lesson player header, placeholder overlay for missing video

To add a new image: copy the file into `public/images/`, then add `image: '/images/your-file.png'` to the lesson in `radar.ts`.

Currently mapped:
- `/images/main-radar-thumb.png` → RADAR course card + overview hero
- `/images/mindset.png` → lesson-1-1 (RADAR Mindset video)
- `/images/radar-flashcards-mindset.png` → lesson-1-3 (Flashcard Drill)
- All other lessons → colored gradient placeholder (auto-generated by type)

## Adding a Video ID
Find the lesson in `src/data/courses/radar.ts` and set `vimeoId: 'YOUR_ID'` in the content.
Lessons without a vimeoId show a "Coming soon" placeholder.
